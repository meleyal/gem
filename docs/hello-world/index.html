Hello World

This describes the basics of sending a note to a DAW via the Web MIDI API on macOS.

First, we need to ensure that macOS' MIDI system is enabled. This is configured in the Audio Midi Setup.app. Go to Window > Show MIDI Studio, double-click IAC Driver, and make sure Device is online is checked. With the default settings, there should be a single Bus 1 under Ports.

Next, we'll need an app that can receive MIDI and generate sounds. I'm using Logic, but Live, Reason, or Reaper will work too. For debugging, a MIDI logger such as MIDI Monitor is also useful.

Next, we need to configure the app to recieve our MIDI messages. The following is specific to Logic Pro X (for other apps, Google "multi track midi routing")

Create a new project with a single software instrument channel (e.g. a Piano). Now, Under Preferences > MIDI > Inputs check IAC Driver Bus 1. Then under File > Project Settings > Recording check Auto demix by channel if multitrack recording.

With that done, we're ready to send some MIDI. Wherever you can run JS in the browser (the console, a file, a script tag), enter the following.

requestMIDIAccess returns a Promise that we need to await, so we wrap everything in an async function.

export default function(word) { console.log(word); }

(async () => { const midi = await navigator.requestMIDIAccess();

// We're only interested in a single output (the IAC Bus we just configured).
const output = midi.outputs.values().next().value; console.log(output.name); //
'IAC Driver Bus 1'

// A single note: middle C at full velocity on MIDI channel 1. output.send([144,
60, 127]);

// You'll often see messages specified using hex notation (an old MIDI
convention). let note = 60; // middle C let noteOn = 0x90; // 144 = channel 1
note on let noteOff = 0x80; // 128 = channel 1 note off let velocity = 0x7f; //
127 = maximum velocity let length = 1000; // 1 second = 1 beat at 60 bpm

// A single note is actually 2 messages, a note on (`NON`), equivalent to //
pressing a key on the piano, and a note off (`NOF`), equivalent to releasing //
the key. output.send([noteOn, note, velocity]); output.send([noteOff, note,
velocity], window.performance.now() + length); })();
